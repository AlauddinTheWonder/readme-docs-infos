How to install and run PS3 games using RPCS3 emulator
====================================================================

Quick Start Guide: https://rpcs3.net/quickstart


1. Download RPCS3 emulator and install.
	https://rpcs3.net/download
	- Download windows compatible file and extract is using 7z software.

2. Download latest PS3 Firmware
	https://www.playstation.com/en-us/support/hardware/ps3/system-software/
	- Right click on "Download PS3 Update" button and click on "Save link as" option to download it.
	- It will download "PS3UPDAT.PUP" file.

3. Download Microsoft VC_redist software and install.
	https://aka.ms/vs/16/release/VC_redist.x64.exe

4. Install and update PS3 firmware in RPCS3.
	- Goto extracted folder of RPCS3 and run "rpcs3.exe"
	- Goto "File -> Install Firmware" and select download "PS#UPDAT.PUP" file.
	- It will install and update the firmware.

5. Steps to download and dump/compile games.
	- Download compatible games from
		- https://psndl.net/packages/17/download

	- Choose game and download two files i.e. ".pkg" and ".rap" file.
	- Goto "File -> Install Packages/Raps/Edats" and select ".rap" file.
	- Again Goto "File -> Install Packages/Raps/Edats" and select ".pkg" file. It will install and compile the game.
	- You should see the compiled game in the list now. Select that and Right-Click and select "Boot" option to start the game.
